# Forrest Field Enhancement Society – Official Website
Website: https://forrestfieldes.ca  

This repository contains the source code for the **Forrest Field Enhancement Society (FFES)** website.  
The society is based in **Ladysmith, British Columbia**, and is focused on community‑led improvements to Forrest Field.

---

## ✅ Purpose of the Society

According to the current site metadata, the FFES is focused on:

- Raising funds to enhance **Forrest Field**  
- Replacing **aging playground equipment** (current top-priority project)  
- Supporting future enhancement phases of the Lot 108 plan  
[1](https://github.com/samdaysecurity/forrestfield-website/blob/main/forrestfieldes2.html)

Upcoming initiatives mentioned in the site’s HTML include:

- A **new major raffle launch** (boat raffle)  
[1](https://github.com/samdaysecurity/forrestfield-website/blob/main/forrestfieldes2.html)

---

## ✅ About This Website

This website provides:

- Public information about the Forrest Field Enhancement Society  
- Updates on current and future improvement projects  
- Images related to Field Phase 2, playground upgrades, and outdoor fitness equipment  
- A community‑accessible source of information hosted on GitHub Pages and served through **forrestfieldes.ca**

The site is a *single‑page*, lightweight, mobile‑responsive HTML layout.

## ✅ Deployment Overview

### **GitHub Pages**
The repo is currently deployed through GitHub Pages, and the custom domain **forrestfieldes.ca** points to it.

Steps to update:

1. Commit changes to `main`  
2. GitHub Pages auto‑builds  
3. Site updates live at https://forrestfieldes.ca

### **Azure Static Web App (Optional Future Upgrade)**

If you choose to switch later, Azure Static Web Apps provides:

- Global CDN  
- Free SSL  
- GitHub‑connected CI/CD  
- Custom domains  

(You are already set up to do Static Web Apps easily if needed.)
